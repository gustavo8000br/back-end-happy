# Back-end Happy

___

## Passos necessários para executar o codigo

* Criar arquivo ``.env`` na raiz com as segintes informações:
**mudar as informações conforme a nescessidade.**

 ```.env
PORT=3333
HOST=localhost
 ```

* Executar no terminal, na pasta raiz do projeto:

```bash
yarn install
```
